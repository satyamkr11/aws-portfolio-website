# AWS Portfolio Website

A personal portfolio website deployed on AWS EC2 using Ubuntu Linux and Apache2.

## Technologies Used

* AWS EC2
* Ubuntu Linux
* Apache2
* HTML
* CSS

## Features

* Personal portfolio website
* Skills section
* Project showcase
* GitHub and LinkedIn integration
* Hosted on AWS cloud infrastructure

## What I Learned

* Launching and configuring EC2 instances
* Linux command-line basics
* Apache2 web server management
* Website deployment on AWS
* Managing website files using Nano editor
